O QUE É FRONT-END

Front-end é toda parte da programação relativa à interface de uma aplicação. Para que fique mais claro, imagine que você acessou o site de sua rede social favorita.

O QUE É BACK-END?

O desenvolvimento back-end cuida das engrenagens de uma aplicação web, criando códigos para que as funções do site sejam executadas. Como o próprio nome sugere, é um trabalho de bastidores.

O QUE É FULL STACK?

Como sua própria denominação indica, o desenvolvedor full stack é o profissional habilitado para compreender e operar em todas as camadas do desenvolvimento de um projeto
